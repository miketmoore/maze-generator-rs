# Maze Generator - Rust Implementation

This is a [Rust](https://www.rust-lang.org/) port of my TypeScript maze generator, found [here](https://github.com/miketmoore/maze-generator-js).

## Run

```
cargo run
```

## Tests

```
cargo test
```
